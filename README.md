# OSINT P

**OSINT P** — мобильное приложение на Flutter для структурированного сбора и анализа разведывательных данных в формате «дел» (cases). Разработано для удобной работы с целями, связями, доказательствами и геоданными в рамках одного инструмента.

---

## Возможности

### Дела (Cases)
- Создание неограниченного числа дел
- Переименование и удаление дел
- Теги для каждого дела *(экспериментально)*
- Генерация PDF всего дела *(экспериментально)*

### Цели (Targets)
- Карточка цели: имя, фамилия, отчество, заметки, теги
- Приоритеты: Высокий / Средний / Низкий *(экспериментально)*
- Четыре вкладки внутри цели: **Инфо**, **Связи**, **Доказательства**, **Карта**

### Инфо — категории и поля
- Произвольные категории с парами ключ–значение
- Перетаскивание для сортировки полей и категорий
- Умное копирование: автоматическое распознавание телефонов, номеров карт (16 цифр), координат
- Настраиваемые символы-пометки для быстрого копирования

### Связи (Connections)
- Привязка целей друг к другу внутри дела с указанием причин
- Граф связей — визуальная схема всех связей с фильтрацией по причине
- Экспорт графа в PNG *(экспериментально)*

### Доказательства (Evidence)
- Прикрепление произвольных файлов (изображения и другие форматы)
- Описание к каждому доказательству
- Изображения включаются в PDF-отчёт

### Карта (Map)
- Автоматическое распознавание координат в полях цели
- Встроенная карта на базе `flutter_map`
- Общая карта всего дела с кластеризацией маркеров *(экспериментально)*
- Экспорт карты в PNG *(экспериментально)*

### Поиск
- Нечёткий (fuzzy) поиск по всем полям, категориям, тегам и заметкам
- Отображение места совпадения в результатах

### Экспорт и импорт
- Экспорт / импорт базы данных в формате JSON
- Экспорт базы SQLite (`.db`)
- Генерация PDF-отчёта по цели (с доказательствами и связями)
- Шаринг PDF и файлов доказательств

### Настройки
- Три темы: Светлая / Тёмная / AMOLED
- Два языка: Русский / English
- Управление пользовательскими пометками копирования
- Экспериментальные функции (включаются по желанию)
- Сброс базы данных

## Технологический стек

| Технология | Назначение |
|---|---|
| Flutter | Кроссплатформенный UI |
| SQLite (`sqflite`) | Локальное хранилище данных |
| `flutter_map` + `latlong2` | Встроенные карты |
| `pdf` + `printing` | Генерация и предпросмотр PDF |
| `file_picker` | Выбор файлов-доказательств |
| `share_plus` | Шаринг файлов |
| `shared_preferences` | Хранение настроек |
| `path_provider` | Доступ к директориям устройства |
| `uuid` | Генерация уникальных идентификаторов |

## Установка и запуск

1. Убедитесь, что установлен [Flutter SDK](https://flutter.dev/docs/get-started/install)
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/osint-v.git
   cd osint-v
   ```
3. Установите зависимости:
   ```bash
   flutter pub get
   ```
4. Запустите приложение:
   ```bash
   flutter run
   ```

> Приложение предназначено для Android и iOS. Все данные хранятся **локально на устройстве** и не передаются на внешние серверы.
